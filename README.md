# BASELINE\_JPEG\_DECODER

A BASELINE JPEG DECODER WHICH SUPPORTS CHROMA SUBSAMPLING WRITTEN ENTIRELY IN C ALONG WITH VALID BASELINE JPEG TEST DATA.



THE PROJECT UPLOADED HERE IS A JPEG DECODER USING DCT FOR BASELINE JPEG IMAGES. THE DECODER CONVERTS THE JPEG FILE INTO .bmp FILE AND THEN OPENS AND SAVES INTO YOUR COMPUTER.



**HOW TO RUN IT??**



DOWNLOAD AND EXTRACT THE REPOSITORY ON ANY FOLDER. THEN OPEN TERMINAL INTO YOUR CURRENT DIRECTORY AND WRITE
**gcc jpeg.c -o decode**



THE CODE WILL START RUNNING AND THE DECODED BMP FILE WILL BE DISPLAYED AND SAVED ONTO YOUR COMPUTER. NOTE THE DISPLAY OF BMP IMAGE IS ONLY SUPPORTED ON WINDOWS AS IT USES

**#else**

&#x20;   **system("xdg-open output.bmp 2>/dev/null || open output.bmp 2>/dev/null");**

**#endif**
